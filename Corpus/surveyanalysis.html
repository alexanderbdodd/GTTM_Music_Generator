Comparison of average preference choices for the difference data sets.

Extracting different data sets has been used to identify where there are clear differences in choice in preferred fragments based on musical experience.

My first approach was to compare the average percentage of choices for the system with the highest rated fragment to see if there was any marked difference between the sets. This was done by calculating the average number of people that chose the highest rated fragment of each pair. Each average was then multiplied by the mean preference scale choice in order to calculate the weight average number of people that preferred a fragment. (i.e., if in pair 1, 10 people chose the highest rated fragment, and the mean value of their choice on the preference scale (from 1 - 3) was 2, then I would multiply 10 by 2).

Pearson's Correlation Test

Pearson's correlation test has been used to assess whether a correlation exists between a fragment receiving a higher percentage of votes if it's a higher rating fragment. This was done for percentages based with and without the preference scale weightings.

The percentage values were correlated against a fragment value, which was either of value 1 if it represented the fragment which received the highest rating of the pair, and a value of -1 if it represented a fragment with the lower rating of the pair.

Preference Reasons Analysis

SAS was used to ascertain whether there were any positive or negative correlations between the reasons respondents gave for preferring a fragment. For example, is there a correlation between respondents finding a piece of music 'more natural' and finding it to be 'more musical'? This was done separately for the set of responses which chose the higher rated fragment and the set of responses which chose the lower rated fragment.
