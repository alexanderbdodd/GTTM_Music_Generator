Manipulators Package

The Player package is an adaptor package for the JMusic library. It provides classes that are able to convert Music Generator classes into JMusic classes, and further provides the ability to create audible representations of objects from the Elements package.

Classes

  1. IntervalEnum enum-type

    Enumerates the different kinds of interval distances supported by the program.

  2. IntervalConstructor class

    This is used to assess interval distances between Key objects, and to find Key objects at given intervals from a given Key object.

  3. ChordConstructor class

    Constructs chords of Key objects according to the given ChordEnum and Key object.

  4. ChordEnum enum-type

    Constrains the type of chord type that is supported by the program.

  5. GeneralManipulator class

    Provides general methods for use by other classes of the Manipulators package.

  7. ScaleModeEnum enum-type

    Enumerates the types of scale supported by the program.

    8. ScaleConstructor class

    Used to generate objects of the Scale class for use in representing different tonal scales.
