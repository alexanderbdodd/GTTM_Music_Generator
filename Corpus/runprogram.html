Running the Program

This document provides an overview of running the program.

The program has been coded using the JAVA language and requires JRE 7 or higher to run. At present there is no front-end GUI.

The program comes as a NetBeans project and can be opened and run using the NetBeans IDE.

Generating Musical Pairs

  1. To run the program, the main method of the Main class (located in the default package) should be run. This will make the program generate two pieces of music which are based on one grammar.
  2. Stage A. The program will start a process of generating a grammar structure and then attempt to find a solution to the grammar. If no solution is found another grammar structure is generated.
  3. Stage B. When the program finds a solution to a grammar structure, it will report this to the operator. A visual representation of the grammar is displayed. (Click here for an example grammar illustration output.)
  4. Stage C. Details about the state of the system that generated the program is logged for future reference.
  5. Stage D. The ratings generator is used to generate ratings concerning how well the solution fits the grammar that was used to construct it. These are reported to the user.
  6. Stage E. The program plays the generated solution and saves the solution as a .midi file.

    The files are given the following naming format:

    year-month-day-hours-minutes-seconds-overallrating.midi (yyyy-mm-dd-hh-mm-ss-overallrating.midi)
