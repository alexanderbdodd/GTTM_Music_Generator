Phrase_Generator Package

The Phrase_Generator package provides classes to be used in generating musical phrase solutions to phrase-length grammars (the music generation process is explained here).

Classes

The Phrase_Generator Package contains the following classes:
  1. Generator Abstract Class

    The Generator class provides methods for extract separate grammar structures from a GrammarConstructor object and get methods to access them. The class is designed to be extended by specific generator classes.

  2. PhraseGenerator Class

    The PhraseGenerator class is used to generate musical phrase solutions to a phrase-structure grammar.

  3. PitchGenerator Class

    The PitchGenerator class is used to generate pitch solutions which fit the relevant grouping structure, prolongational reduction, and time-span reduction structure constraints.

  4. DurationsGenerator Class

    The DurationsGenerator class is used to produce rhythm solutions to the pitch solutions based on the relevant metrical structure and grouping structure constraints.

  5. NoteGenerator Class

    The NoteGenerator class provides methods for randomly generating different musical elements including AttackEvents, Scales, and Keys.

  6. HomophonyConstructor Class

    An incomplete class that is supposed to provide homophonic solutions to the monophonic solutions produced by the PhraseGenerator according to GTTM rules constraints. This class is incomplete and not useable at present.
