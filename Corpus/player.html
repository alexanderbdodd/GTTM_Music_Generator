Player Package

The Player package is an adaptor package for the JMusic library. It provides classes that are able to convert Music Generator classes into JMusic classes, and further provides the ability to create audible representations of objects from the Elements package.

Classes

  1. JMusic Adaptor Class

    A class that is used to convert various instances of classes from the Elements package into instances of classes from the JMusic library.

  2. Player class

    Provides methods for playing EventStream objects and writing them out at .midi files.
