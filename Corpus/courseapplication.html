How the Music Generator Program makes use of Course Related Concepts

This project has been undertaken as part of a MSc Computer Science Conversion course. In this document I will give an overview as to how I have made use of the skills I have developed throughout the course.

  1. Use of Recursion

    Recursion was a programming technique that I learnt as part of the Logic and Logic Programming course in the second semester, and I have used the technique several times in the development of my program. This technique lets the program run sequentially through all possible combinations of certain elements, and has been particularly important in constructing what I term 'pitch-chains', which are candidate chains of pitches which are then assessed as to their suitability in fitting a particular grammar. Recursion was used to run through all possible pitches in each position of the chain, such that all possible combinations were assessed.

  2. Use of Java Swing and Java2D

    Java Swing and Java2D were taught as part of the Advanced Java course in the first semester. I have made use of these APIs to develop the GrammarIllustrator module which is used to create visual representations of grammar structures.

  3. Use of the Singleton Design Patterns

    Coding design patterns were covered as part of the Advanced Java course, and I have made use of a the singleton design pattern within various classes to reduce the need to create unncessary multiple instances of objects. This pattern is particularly used in the GTTM_Analyser modules.

  4. Unit Testing

    Unit testing was covered as part of the Software Engineering course and I have made use of unit tests to check that my classes behave in the expected way.

  5. Use of Polymorphism

    Polymorphism was covered in the Advanced Java module, and its use can be seen in the way objects extending the Event class are used, as well as the classes extending the Key class.

  6. Encapsulation

    This was taught as part of the Advanced Java module. The program is written in object-oriented code and has many classes that are designed to encapsulate one musical element. For instance, and instance of the AttackEvent class encapsulates details about one pitch and its duration, and an instance of ProlongationalBranch encapsulates details of one single branch of a prolongational reduction.

  7. Use of Exception classes

    The throwing of Exceptions was taught as part of the Advanced Java course and have been used to indicate errors in such things as grammar construction or analysis errors.
