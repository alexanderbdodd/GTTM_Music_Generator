Examples generated from different ratings

The system makes use of different weights for different ratings to find weighted averages for rating values. I haven't had enough time to systematically test out different weightings systems, but I have done a brief, informal testing of different ratings. The results can be found in this folder.

Grammar Illustration Examples

As well as creating musical fragments, the program also generates grammar structures. The GrammarIllustrator class is used to generate visual representations of the generated grammar structures. Example illustrations can be viewed here.
